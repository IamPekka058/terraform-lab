# Terraform Lab
This repository is my personal lab for learning Terraform. It contains projects and modules that I have created to practice and experiment with Terraform's capabilities.

## Modules
- **azure-budget**: This module is designed to create monthly budgets in Azure. It allows you to set up budget thresholds and receive notifications when those thresholds are exceeded.

## Projects
- **azure-kuma**: This project is focused on deploying a Kuma Container behind EntraID. It showcases how to set up and manage a Kuma Container in an Azure environment, ensuring secure access through EntraID.
- **azure-budget**: This project utilizes the 'azure-budget' module to create and manage budgets in Azure. It demonstrates how to implement the module and configure it according to specific requirements.
